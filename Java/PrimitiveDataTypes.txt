        int intValue = 2147483647;

        System.out.println("Double Value: " + doubleValue);
        System.out.println("Char Value: " + charValue);

In this program, we declare variables of various primitive data types and assign them some sample values. Here's a breakdown of the primitive data types used:

Integer types:

byte: 8-bit signed integer, ranging from -128 to 127.
short: 16-bit signed integer, ranging from -32,768 to 32,767.
int: 32-bit signed integer, ranging from -2,147,483,648 to 2,147,483,647.
long: 64-bit signed integer, ranging from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807.
Floating-point types:

float: 32-bit floating-point number, ranging from approximately ±1.4 x 10^-45 to ±3.4 x 10^38 with a precision of 7 digits.
double: 64-bit floating-point number, ranging from approximately ±4.9 x 10^-324 to ±1.8 x 10^308 with a precision of 15 digits.
Character type:

char: 16-bit Unicode character, representing a single character.
Boolean type:

boolean: Represents a truth value, either true or false.
